# 🎵 MP3 Player на GTK4

Простой музыкальный плеер с графическим интерфейсом на **GTK4** и аудиодвижком **SDL2_mixer**. Поддерживает воспроизведение MP3, плейлисты из папок, перемотку, громкость, перемешивание и режимы повтора.

## ✨ Возможности

- Воспроизведение MP3 (через SDL2_mixer и libmpg123)
- Плейлист на основе выбранной папки (сканирование `.mp3`)
- Перемотка трека (ползунок)
- Регулировка громкости (ползунок)
- Пауза/возобновление (кнопка или пробел)
- Предыдущий/Следующий трек (кнопки или клавиши)
- Режим **перемешивания** (shuffle)
- Режим **повтора одного трека**
- Режим **зацикливания одного трека** (бесконечный повтор)
- Автоматический переход к следующему треку по окончании
- Обновление плейлиста из папки
- Выбор папки через диалог (GTK FileChooser)
- Отображение текущего времени и общей длительности трека

---

## 📦 Зависимости

Для сборки необходимы:

- **gcc** (компилятор C)
- **make** (система сборки)
- **pkg-config** (для поиска библиотек)
- **GTK4** (≥ 4.10, рекомендуется 4.14+)
- **SDL2** (libsdl2)
- **SDL2_mixer** (libsdl2-mixer)
- **libmpg123** (декодер MP3)

---

## 🛠️ Установка зависимостей

### Ubuntu / Debian (20.04+)
```bash
sudo apt update
sudo apt install build-essential libgtk-4-dev libsdl2-dev libsdl2-mixer-dev libmpg123-dev
```
## 💡 Проверка зависимостей:
В корне проекта есть скрипт check-deps.sh. Запустите его, чтобы убедиться, что все библиотеки установлены:
```bash
./check-deps.sh
```
## 🚀 Сборка и запуск

1.Клонируйте репозиторий:
```bash
git clone https://github.com/ваш-логин/MP3_Player.git
cd MP3_Player
```
2.Соберите проект:
```bash
make
```
Исполняемый файл появится в папке build под именем app.

3.Запустите плеер:
```bash
./app
```
